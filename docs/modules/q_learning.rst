Q-Learning
==========

`Q-Learning <https://en.wikipedia.org/wiki/Q-learning>`_ is a model-free reinforcement learning algorithm to learn a policy telling an agent what action to take under what circumstances. It does not require a model of the environment, and it can handle problems with stochastic transitions and rewards, without requiring adaptations..
